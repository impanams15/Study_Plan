# Study_Plan

## AI Study Planner

A comprehensive web application designed to help students organize their study schedules efficiently using AI-driven logic. This tool allows users to input their available study hours and subjects with priority levels, automatically generating a balanced timetable.

### Key Features

- **Smart Timetable Generation:** Automatically allocates study hours based on subject priorities (High, Medium, Low).
- **User Authentication:** Secure signup and login system with local storage persistence.
- **Profile Management:** Users can update their profiles and view their last login time.
- **Productivity Hub:**
  - **Pomodoro Timer:** Built-in timer for focused study sessions.
  - **Progress Tracker:** Visual progress bar to track daily study goals.
  - **Smart Tips:** Context-aware study tips based on your subjects.
  - **Daily Motivation:** Random motivational quotes to keep you inspired.
- **Data Persistence:** Saves your timetable and inputs locally so you can pick up where you left off.
- **Admin Dashboard:** A simple interface to view registered users.
- **Dark/Light Mode:** Toggle between themes for comfortable viewing.
- **Export Options:** Export your timetable to CSV or save it to your profile for later reference.

### Technologies Used

- HTML5
- CSS3 (Modern Flexbox & Grid, Glassmorphism UI)
- Vanilla JavaScript (ES6+)
- LocalStorage for data management

### How to Run

Simply open `login.html` in your web browser to start using the application. No backend server is required as it runs entirely in the browser.
